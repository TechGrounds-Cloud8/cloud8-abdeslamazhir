# Variables







## Exercise 1


- Create a new script.
- Create two variables x and y. Assign a numerical value to both variables.
- Print the values of x and y.
- Create a third variable named z. The value of z should be the sum of x and y.
- Print the value of z.

## Result 
Script:

    x = 5
    print (x)
    y = 20
    print (y)
    z = x + y
    print (z)
